"Ok, I see many of you that have flikr links on your blogs. How do you
do that?"

You create a flickr badge with an HTML code generator (because I don't
know HTML) like this: http://flagrantdisregard.com/flickr/badge.php
Then insert it into your "format" at flickr where you want it. Took me
a while to figure out where to insert the code so it made sense in the
layout.

Here's the one I used: http://www.flickr.com/badge_new.gne
It's the rotating "flash badge", which looks really cool.
Here's a blog that describes the HTML coding used to create such a
badge: http://veerle.duoh.com/blog/comments/fickr_badge_w3c_valid/

Cool stuff!
