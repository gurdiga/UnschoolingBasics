I just wanted to let you all know that the unschooling.info message
boards ARE still being worked on and we will get them back.

The server had a major hardware failure. Everything has been
transferred to a new (and, they say, more reliable) server.

But the forum code (the program that makes the message boards work)
has been "corrupted." We think we can get all the past message back -
they're all there and not impacted, but the registered member list is
messed up.

So - we have to figure out how to re-upload the forum code and get it
to find the old messages. Once that is done, people will probably
have to register again if they want to post.

Rue Kream's DEAR husband, Jon, is working on this for us. So - please
be patient and grateful.

Hang in there ----

-Pam Sorooshian
