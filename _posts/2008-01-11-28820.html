Amie,



I'm a newbie here, but my hubby works for Blizzard (who makes WoW), so I
asked him about this. It is illegal to run a private server for WoW. There
is also a chance (and this happens all the time, actually), that the person
running the private server would hack your system. In addition, there may
not be (and probably aren't) backups for your data, any support for
problems, updates to the software (which often happens every few weeks),
etc. Personally, sounds like a lot of frustration potential to me! But I
can see the draw in playing on a private server anyway, so long as you are
aware of the risks.
